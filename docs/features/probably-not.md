# Probably Not

Considered and deprioritised. Kept here to avoid re-litigating.

## Applications

- **Font browser** — installed-font preview grid
- **Wallpaper picker** — `swww` browser; no wallpapers in use

## Plugins

### Bar

- **Pomodoro** — countdown timer; IPC controllable
- **Weather** — current conditions from `wttr.in`; forecast dropdown
- **Night light** — toggle `wlsunset`/`gammastep` with temp slider
- **NixOS update notifier** — badge when system is behind upstream
- **Keyboard layout switcher** — cycle via `hyprctl switchxkblayout`
- **GitHub/GitLab notifications** — unread badge via API
- **Crypto/stock ticker**
- **Git branch indicator** — unclear what "focused window's repo" means outside a terminal
- **Clock timestamp copy** — too niche; widget action without visual feedback is confusing

### Launcher

- **Calculator** — evaluate expressions; copy result
- **Recent files** — fuzzy search `recently-used.xbel`
- **Password generator**
- **IDE project picker** — terminal workflow already covers this
- **Dictionary** — search engine covers the need

### OSD

- **OsdCapsLock**/**OsdNumLock** — not worth the screen noise
- **OsdPowerProfile** — infrequent, visible in the bar already
- **OsdColourTemperature** — gradual transitions; OSD more disruptive than the change
- **OsdNowPlaying** — bar's MediaPlayer covers it
